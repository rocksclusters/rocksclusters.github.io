2.1. Adding the Roll

The Area51 Roll must be installed during the Frontend installation step of your cluster (refer to section 1.2 of the Rocks usersguide).

The Area51 Roll is added to a Frontend installation in exactly the same manner as the required HPC Roll. Specifically, select the Area51 Roll in exactly the same way as you select any other roll
