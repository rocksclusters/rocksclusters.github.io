9.15. swap

9.15.1. swap host interface

rocks swap host interface {host} [ifaces=string] [sync-config=boolean]

Swaps two host interfaces in the database.

arguments

host

Host name of machine

parameters

[ifaces=string]

Two comma-separated interface names (e.g., ifaces="eth0,eth1").

[sync-config=boolean]

If "yes", then run 'rocks sync config' at the end of the command. The default is: yes.
