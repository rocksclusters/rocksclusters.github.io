4.3. Adding Applications to Compute Nodes

If you have code you'd like to share among the compute nodes, but your code isn't in an RPM (or in a roll), then this procedure describes how you can share it with NFS.

On the frontend, go to the directory /share/apps.

# cd /share/apps

Then add the files you'd like to share within this directory.

All files will also be available on the compute nodes under: /share/apps. For example:

# cd /share/apps
# touch myapp
# ssh compute-0-0
# cd /share/apps
# ls
myapp
