9.14. save

9.14.1. save host partitions

rocks save host partitions

Save the frontend current mounted paritions into the Rocks database. You can use this command to update partition information when you change you frontend partitions configuration.

examples

# rocks save host partitions

Save the current mounted partition into rocks database
