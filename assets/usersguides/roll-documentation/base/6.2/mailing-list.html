6.3. Email Discussion List

The primary mode for support for Rocks is through our email discussion list.

Click here to subscribe.

The archives hold messages from the Rocks discussion list dating back to June 2002.

If you need to search old messages you can use the following links:

  • Mailing list ARChives

  • Google with the extra directive site://lists.sdsc.edu/pipermail/npaci-rocks-discussion in the search bar (to limit search results to Rocks mailing list only).
