3.5. enable

3.5.1. enable roll

rocks enable roll [roll...]

Enable an available roll. The roll must already be copied on the system using the command "rocks add roll".

arguments

[roll]

List of rolls. This should be the roll base name (e.g., base, hpc, kernel). If no rolls are listed, then all the available rolls are enabled.

examples

# rocks enable roll kernel

Enable the kernel roll

# rocks enable roll

Enable all the available rolls

related commands

add roll

create roll

disable roll

list roll
