3.8. Using MrBayes

3.8.1. About

MrBayes is a program used for bayesian inference of phylogeny. MrBayes is cowritten by John Huelsenbeck and Fredrik Ronquist.

3.8.2. Usage

MrBayes uses the NEXUS file format for input. To use MrBayes, just type mb at the command line

[nostromo@rocks-168 mrbayes]$ mb
