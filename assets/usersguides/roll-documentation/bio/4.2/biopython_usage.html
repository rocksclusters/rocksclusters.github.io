3.13. Using Biopython

3.13.1. About

Biopython is a set of python modules for Bio-informatics computation.

3.13.2. Usage

Biopython modules can be used to supplement already existing applications such as blast. For information on how to use the library, please refer to the biopython documentation.

3.13.3. Further Information

Further information about biopython is available at the Biopython home page
