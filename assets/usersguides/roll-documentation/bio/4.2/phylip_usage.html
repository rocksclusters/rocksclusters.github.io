3.9. Using Phylip

3.9.1. About

Phylip - Phylogeny Inference Package - is a package of programs for inferring phylogenies or evolutionary trees. The version distributed with Rocks is v3.65.

3.9.2. Further Information

Furtherinformation about Phylip is available at the Phylip home page.
