3.6. Glimmer

3.6.1. About

Glimmer is a system for finding genes in microbial DNA, especially the genomes of bacteria, archaea, and viruses. Glimmer was developed at the Centre for BioInformatics and Computational Biology. The version that is distributed with Rocks is Glimmer v3.02.

3.6.2. Using Glimmer

Glimmer is installed at /opt/Bio/glimmer/. Glimmer is run in 2 stages.

  • Glimmer is trained on a particular training set of similar species to recognize genes

  • Glimmer is then run on an input DNA sequence to find genes

3.6.3. Further Information

Further information about the usage of Glimmer can be found in the release notes of the software, available here. This file is also available on the frontend of your cluster at /opt/Bio/glimmer/glim302notes.pdf
