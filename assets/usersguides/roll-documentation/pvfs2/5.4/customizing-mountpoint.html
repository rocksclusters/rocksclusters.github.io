4.1. Changing the PVFS2 Mountpoint

The default PVFS2 mountpoint is /mnt/pvfs2. To change it, edit the file /opt/pvfs2/conf/variables, and change the value:

PVFS2_MOUNTPOINT=/mnt/pvfs2

For example, to change the mountpoint to /my/mount, the line above should look like:

PVFS2_MOUNTPOINT=/my/mount

After you modify the file above, you must apply the change to the cluster -- see the section Applying the Configuration for details.
