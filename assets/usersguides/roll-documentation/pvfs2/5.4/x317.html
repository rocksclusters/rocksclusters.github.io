5.3. remove

5.3.1. remove host pvfs partition

rocks remove host pvfs partition {host}

Remove the partition that is associated with a PVFS2 server. This partition is the location on the PVFS2 server that is used to store data blocks. This command simply removes the partition configuration from the frontend's database. It does not remove data on the PVFS2 server.

arguments

host

One host name.

examples

# rocks remove host pvfs partition pvfs2-io-0-0

Remove the partition entry in the database for pvfs2-io-0-0.
