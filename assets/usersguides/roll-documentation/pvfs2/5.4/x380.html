5.6. sync

5.6.1. sync pvfs

rocks sync pvfs

Generates new PVFS2 configuration files for each PVFS2 storage server, then reconfigures the PVFS2 storage system. This command also remounts the PVFS2 file system on all clients.

examples

# rocks sync pvfs
