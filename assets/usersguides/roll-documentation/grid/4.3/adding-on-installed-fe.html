2.3. On an Existing Server

The grid Roll may also be added onto an existing server (or frontend). For sake of discussion, assume that you have an iso image of the roll called grid.iso. The following procedure will install the Roll, and after the server reboots the Roll should be fully installed and configured. 
$ su - root
# mount -o loop grid.iso /mnt/cdrom
# cd /home/install
# rocks-dist --install copyroll
# umount /mnt/cdrom
# rocks-dist dist
# kroll grid | bash
# init 6
