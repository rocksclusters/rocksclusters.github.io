2.2. Using the Python Roll

The Python roll installs Python versions 2.7.6 and 3.3.4. These are installed in /opt/python.

The Python roll installs Environment Modules files for Python 2.7.6 and 3.3.4. These are installed in /usr/share/Modules/modulefiles.

Numpy 1.8.0 is compiled and installed with Python 2.7.

Tutorials for using Python and its components are available from the Python website. Tutorials for using NumPy are available from the NumPy website.
