2.1. Environment Modules for Python

As of Rocks 6.2 (Sidewinder), Environment Modules are utilized to control Python path names.

  • To see the currently loaded modules:

    % module list
Currently Loaded Modulefiles:
  1) rocks-openmpi
%

  • To see available modules:

    % module avail
------------------------ /usr/share/Modules/modulefiles ------------------------
dot           module-info   null          use.own
module-cvs    modules       opt-python    rocks-openmpi

%

  • To load the opt-python module:

    % module load opt-python
% module list
Currently Loaded Modulefiles:
  1) rocks-openmpi   2) opt-python
