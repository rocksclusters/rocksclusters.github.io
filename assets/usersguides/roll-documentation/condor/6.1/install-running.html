2.3. Install on Running System

The Condor Roll can be installed on a running system. The following assumes that roll is available in .iso form and is called "condor.iso".

$ su - root 
# rocks add roll condor.iso
# rocks enable roll condor
# rocks create distro
# rocks run roll condor | bash
# shutdown -r now
