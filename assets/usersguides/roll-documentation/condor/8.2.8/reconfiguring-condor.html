4.5. Reconfiguring Condor after Installation

The configuration of Condor is done during the install, the resulting configuration files are located in /opt/condor/etc. To reconfigure Condor on a node, make appropriate attribute using the commands above and then run 

# rocks sync host condor <hostname>

This will rewrite the condor_config.local on the file and then calls the Condor command /opt/condor/sbin/condor_reconfigure

To view the contents of the condor_config.local before making changes, use rocks report host condor config <hostname>

To find information about administrating and using Condor Pools please see the original Condor manual at Condor manuals or locally .
