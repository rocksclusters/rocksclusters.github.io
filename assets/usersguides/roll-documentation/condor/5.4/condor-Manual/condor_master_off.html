condor_master_off

Shutdown Condor and the condor_master

Synopsis

condor_master_off [-help] [-version] [hostname ...]

Description

condor_master_off no longer exists.

General Remarks

condor_master_off no longer exists as a Condor command. Instead, use 
        condor_off -master
to accomplish this task.

See Also

See the condor_off manual page.
