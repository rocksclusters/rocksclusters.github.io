condor_reconfig_schedd

Reconfigure condor schedd

Synopsis

condor_reconfig_schedd [-help] [-version] [hostname ...]

Description

condor_reconfig_schedd no longer exists.

General Remarks

condor_reconfig_schedd no longer exists as a Condor command. Instead, use 
        condor_reconfig -schedd
to accomplish this task.

See Also

See the condor_reconfig manual page.
