4.10. resume

4.10.1. resume host vm

rocks resume host vm {host...}

Resume a paused VM slice on a physical node.

arguments

host

A list of one or more VM host names.

examples

# rocks resume host vm compute-0-0-0

Resume paused VM host compute-0-0-0.
