4.6. pause

4.6.1. pause host vm

rocks pause host vm {host...}

Pauses a VM slice on a physical node.

arguments

host

A list of one or more VM host names.

examples

# rocks pause host vm compute-0-0-0

Pause VM host compute-0-0-0.
