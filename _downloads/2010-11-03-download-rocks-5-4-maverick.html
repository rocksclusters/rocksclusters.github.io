Building a Compute Cluster

To build a compute cluster, boot your frontend with the Kernel/Boot Roll. Then supply the Core Roll, OS Roll Disk 1 and OS Roll Disk 2 and the Service Pack Roll. From the Core Roll, be sure to select the Base and Web Server Rolls (these rolls are required). Feel free to select any or all the optional rolls from the Core Roll (Area51, HPC, Ganglia, SGE and Xen).

Building Custom Clusters

If you wish to build a custom cluster, you must choose from our a la carte selection, but make sure to download the required Base, Kernel/Boot, Web Server, Service Pack and all of the OS rolls.

MD5 Checksums

Please double check the MD5 checksums for all the rolls you download.
