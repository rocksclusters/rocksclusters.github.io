Operating System Base

Rocks 6.2 (Sidewinder) is based upon CentOS 6.6 with all updates available as of 10 May 2015.

Building a Compute Cluster

To build a bare-bones cluster, boot your frontend with the Kernel/Boot Roll. Then supply the base Roll, OS Roll Disk 1, OS Roll Disk 2, and OS Disk 3. More complete clusters also select the following rolls: Area51, HPC, Ganglia, web-server, perl, python, and SGE. Torque is an optional replacement for SGE. Condor can be used independently or in conjunction with SGE/Torque. ZFS version 0.6.4.1 can be used to build reliable storage systems.

New for 6.2 is support for perfSONAR (version 3.4.2).

Building Custom Clusters

If you wish to build a custom cluster, you must choose from our a la carte selection, but make sure to download the required Base, Kernel/Boot and all of the OS rolls.

MD5 Checksums

Please double check the MD5 checksums for all the rolls you download.
