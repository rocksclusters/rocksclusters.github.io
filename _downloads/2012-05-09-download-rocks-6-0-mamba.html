Rocks 6.0 (Mamba) Menu

Operating System Base

Rocks 6.0 (Mamba) is based upon CentOS 6.2 with all updates available as of 7 May 2012.

Building a Compute Cluster

To build a bare-bones cluster, boot your frontend with the Kernel/Boot Roll. Then supply the base Roll, OS Roll Disk 1 and OS Roll Disk 2. More complete clusters also select the following rolls: Area51, HPC, Ganglia, web-server, perl, python, and SGE. Torque is an optional replacement for SGE. Condor can be used independently or in conjunction with SGE/Torque

Building Custom Clusters

If you wish to build a custom cluster, you must choose from our a la carte selection, but make sure to download the required Base, Kernel/Boot and all of the OS rolls.

MD5 Checksums

Please double check the MD5 checksums for all the rolls you download.
