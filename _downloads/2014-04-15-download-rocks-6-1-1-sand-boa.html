Operating System Base

Rocks 6.1.1 (Sand Boa) is based upon CentOS 6.5 with all updates available as of 14 April 2014. The OS roll includes an update to
Openssl that addresses the heartbleed vulnerability. This release supports 64-bit systems only.

Building a Compute Cluster

To build a bare-bones cluster, boot your frontend with the Kernel/Boot Roll. Then supply the base Roll, OS Roll Disk 1, OS Roll Disk 2, and OS Disk 3. More complete clusters also select the following rolls: Area51, HPC, Ganglia, web-server, perl, python, and SGE. Torque is an optional replacement for SGE. Condor can be used independently or in conjunction with SGE/Torque. New for 6.1.1 is the Fingerprint roll which can help in moving compiled applications to/from a Rocks cluster .

Building Custom Clusters

If you wish to build a custom cluster, you must choose from our a la carte selection, but make sure to download the required Base, Kernel/Boot and all of the OS rolls.

MD5 Checksums
