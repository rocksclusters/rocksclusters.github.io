Operating System Base

Rocks 6.1 (Emerald Boa) is based upon CentOS 6.3 with all updates available as of 27 Nov 2012.

Building a Compute Cluster

To build a bare-bones cluster, boot your frontend with the Kernel/Boot Roll. Then supply the base Roll, OS Roll Disk 1 and OS Roll Disk 2. More complete clusters also select the following rolls: Area51, HPC, Ganglia, web-server, perl, python, and SGE. Torque is an optional replacement for SGE. Condor can be used independently or in conjunction with SGE/Torque. New for 6.1 is the ZFS roll. This is a source roll and must be rebuilt on your system to comply with licensing.

Building Custom Clusters

If you wish to build a custom cluster, you must choose from our a la carte selection, but make sure to download the required Base, Kernel/Boot and all of the OS rolls.

MD5 Checksums

Please double check the MD5 checksums for all the rolls you download.
