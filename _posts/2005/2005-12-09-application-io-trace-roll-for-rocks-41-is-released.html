The App Trace Roll for i386 and x86_64 is released.

The App Trace Roll installs and configures system tracing tools that records the size and offset of every read/write system call for all processes that are running on cluster nodes.

The preface has an overview and defines the purpose of the roll.

Download

  • App Trace Roll for i386
  • App Trace Roll for x86_64

Roll Documentation

In the App Trace Roll Users' Guide, you'll find info on how to:

  • Install the roll (both frontend install and 'on-the-fly' methods are supported)
  • Trace specific applications
  • Collect and analyze the trace files from each compute node
