The Service Pack Roll for i386 and x86_64 architectures is released.

This roll contains a critical fix for all Rocks v4.2.1 clusters. All clusters running Rocks v4.2.1 should install this roll. This roll can be installed onto a running frontend (neither reinstallation nor reboot is required).

Descriptions of the included fixes can be found here.

ISO images for the Service Pack Roll can be found on the Downloads page.
