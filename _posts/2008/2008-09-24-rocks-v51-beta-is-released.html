Rocks v5.1 Beta is released for i386 and x86_64 CPU architectures.

Downloads

  • Jumbo Roll (Area51, Base, Bio, Ganglia, HPC, Java, Kernel, OS disk 1 and 2, SGE, Web Server, Xen) i386 (dvd)
  • Jumbo Roll ((Area51, Base, Bio, Ganglia, HPC, Java, Kernel, OS disk 1 and 2, SGE, Web Server, Xen) x86_64 (dvd)
  • Individual Rolls
  • MD5 checksums

Installation Documentation

Installation documentation can be found here.

If you plan on building virtual clusters, then start here.

There is only one significant change: when the Kernel/Boot CD boots, at the "boot:" prompt, you need to type "build" (not "frontend" as in the past releases).

Notes on One of the New Features

  • Virtual Clusters Support

    • If you include the Xen roll, you will now be able to configure virtual clusters using Rocks.

    Preliminary documentation on how to configure and install virtual clusters can be found here.
