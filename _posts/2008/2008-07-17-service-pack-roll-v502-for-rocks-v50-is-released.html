The Service Pack Roll v5.0.2 for i386 and x86_64 architectures is released.

This roll includes the following new fix:

  • Base Roll - A 411 scalability fix. On larger clusters, 'rocks sync users' was not pushing out all the user-related files (e.g., /etc/shadow, /etc/passwd, /etc/group, etc.). The 411 listener on the compute nodes has been fixed to address this problem.

As well as the fixes from the previous release of the Service Pack Roll:

  • Base Roll - An installation scalability fix.
  • SGE Roll - A fix for the 'Job Queue' web page.

Detailed descriptions of all the fixes can be found here.

This roll can be installed onto a running frontend.

ISO images for the Service Pack Roll can be found on the Downloads page.

Installation instructions can be found here.
