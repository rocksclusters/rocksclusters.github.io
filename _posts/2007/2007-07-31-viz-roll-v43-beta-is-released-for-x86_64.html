An early Beta is being released for the Viz Roll for x86_64 platforms.
This Beta is intended to allow users to verify their 64bit machines are
compatible with Rocks 4.3 and Viz. Currently, only DMX and Chromium are
included. The final release will include SAGE v2 (from EVL) and refreshed
nVidia drivers.

To build a Viz cluster you will need the following Rolls:

  • Kernel / Boot Roll
  • Core Roll
  • All 5 OS CD Rolls (not just the first two)
  • Viz Beta Roll (ftp), (http)

Initial documentation for this Beta is here.
