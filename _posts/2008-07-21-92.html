The Viz Roll for i386 and x86_64 architectures is released.

This release of the Viz Roll includes the following new features:

  • A Rocks-specific version of SAGE 3.0 is now included
  • Enhanced Chromium support for applications like Google Earth
  • Enhanced Bezel/Mullion hiding control via the Rocks command line
