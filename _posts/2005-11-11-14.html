Scalable Systems is pleased to announce the immediate availability of the SCALABLE RxC Roll for Rocks v4.1.

Overview for RxC 1.2.2:

  • This is version 1.2.2 for Scalable Systems Scalable RxC Roll.
  • This release is fully tested and qualified for Rocks 4.1 (Fuji) on i386 and x86-64 clusters.

Highlights:

  • This release provides NFS support.
  • It is compatible with the SUN N1GE Roll.
  • Bugs known since the last release have been fixed.
