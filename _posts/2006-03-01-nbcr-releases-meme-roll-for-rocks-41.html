Nadya Williams from NBCR announces:

The National Biomedical Computation Resource at University of California, San Diego is pleased to announce the availability of MEME Roll package for Rocks clusters version 4.1.

The MEME system provides online tools for discovering and using protein and DNA sequence motifs. The MEME Roll provides MEME deployment and integration on Rocks clusters. More information about MEME can be found here. This MEME Roll
contains MEME version 3.5.2.

The MEME Roll can be downloaded from the Rocks download site or from the NBCR site. Roll documentation including installation and usage information is available at here.

Questions or comments may be directed to Nadya Williams (nadya@sdsc.edu)
